# 🥖 Golden Crust 麵包店庫存管理系統

為現代烘焙坊打造的專業、高效數位解決方案。

## 🌟 核心願景
**Golden Crust 管理系統** 不僅僅是一個品項清單；它是烘焙店員工的「數位大腦」。它將雜亂的手動記錄轉化為簡化且防錯的數位化作業，讓烘焙師能專注於手工藝，而由系統處理數據。

## 🛠️ 技術架構（系統「引擎」）
- **店面（前端）：** 使用原生 JavaScript 與北歐簡約風 CSS 構建。介面乾淨、極簡且具備響應式設計。
- **廚房（後端）：** 由 Python 與 Flask 驅動。這位「主廚」處理所有邏輯，確保數據正確流動。
- **儲藏室（資料庫）：** 使用 SQLite 與 SQLAlchemy，安全且永久地儲存每一筆產品紀錄。

## 🚀 專為店員設計的核心功能

### 1. 永久數位保險箱
不再需要紙本清單。每一條酸種麵包、每一份甜點都儲存在永久資料庫中。即便斷電或重啟，庫存數據依舊安全無虞。

### 2. 高速搜尋與導覽
在超過 50 個品項中尋找特定產品只需一瞬間。透過**即時搜尋**功能，輸入幾個字母就能過濾整個庫存。**分頁功能**則將清單組織成易於閱讀的頁面，保持介面整潔。

### 3. 「預警系統」低庫存邏輯
系統會幫員工盯住數字。當某項產品的庫存降至 **5 份或以下**時，該行會自動變為淡紅色並顯示 **「低庫存」** 標籤，提醒烘焙師接下來該製作哪種產品。

### 4. 防錯使用者體驗 (UX)
透過整合 **SweetAlert2**，系統能與使用者進行親切互動：
- **重複項防止：** 防止意外重複輸入相同的產品名稱。
- **智慧確認：** 在刪除數據前詢問「您確定嗎？」，避免誤刪。
- **成功回饋：** 在任務完成時提供令人滿意的綠色勾勾動畫，增加操作真實感。
